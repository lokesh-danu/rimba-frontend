# Rimba Frontend V2
- This is the new frontend for Rimba. It is built with Next.js and TypeScript.
- It is currently under development and not all features are available.
- It is using the new Rimba V2 API.
- Deployed on Netlify

## Setup

```bash
npm install
```

## Development

```bash
npx next dev
```

## Production

```bash
npm run build
npm run start
```
